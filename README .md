# 魔方求解器

这是一个基于 Android 平台开发的三阶魔方识别与还原工具。项目支持相机常规扫色、基于 YOLOv8-NCNN 的 AI 扫描识别（300张自己的魔方训练的小模型），并内置了轻量化 3D 还原步骤演示，支持分步控制与无级播放速度调节。
